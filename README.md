# Installation
```
pip3 install conolist
```

# Running
from anywhere in your terminal just run `conolist`

# Keys and features available
| Key | Purpose|
| --- | --- |
| q | quit |
| c  | enter command |
| i  | option menu on folders |
| Key up | go up |
| Key Down  | go down |
| Enter on folder | go to that path |
| Enter on file | option menu for file |
